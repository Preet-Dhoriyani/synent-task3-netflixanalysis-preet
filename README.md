# Netflix Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset to uncover trends, patterns, and insights related to Netflix content.

The analysis includes data exploration, missing value analysis, content distribution, country-wise trends, genre analysis, ratings analysis, and content growth over time.

---

## 🎯 Objective

- Perform Exploratory Data Analysis (EDA)
- Generate summary statistics
- Analyze missing values
- Identify trends and patterns
- Visualize important insights using graphs
- Derive meaningful business insights from Netflix content data

---

## 📂 Dataset

Dataset Used:

- netflix_titles.csv

The dataset contains information about Netflix Movies and TV Shows, including:

- Title
- Type
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre

## 📈 Key Insights

1. Movies constitute a larger share of Netflix content compared to TV Shows.

2. The United States contributes the highest amount of content on Netflix.

3. Netflix experienced rapid content growth after 2015.

4. TV-MA is among the most common content ratings.

5. International Movies and Dramas are some of the most popular genres.

6. Several directors have contributed multiple titles to the platform.

7. Content production and release have increased significantly over recent years.
